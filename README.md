# placement-projects
Academic and internship projects submitted for placement verification

## Projects

### 1. Gridworld Reinforcement Learning
- **Type**: Individual Project
- **Description**: Simulates a Gridworld environment and applies classical reinforcement learning algorithms
- **Technologies**: Python, Reinforcement Learning
- **Features**:
  - Gridworld environment design
  - Value iteration and Policy iteration
  - Policy visualization and convergence analysis

### 2. Hidden Markov Model POS Tagging
- **Type**: Individual Project
- **Description**: Implements a Part-of-Speech tagger using a Hidden Markov Model trained on the UD English EWT dataset
- **Technologies**: Python, NLP
- **Features**:
  - HMM-based sequence modeling
  - Viterbi decoding for tag inference
  - Accuracy evaluation and error analysis

### 3. Supermarket Management System
- **Type**: Group Project
- **Description**: Database Systems project combining JavaSwing, JavaFX, and Java SQL
- **Technologies**: Java (97.9%), CSS (2.1%), SQL
- **Original Repository**: [physicsilu/Supermarket-Management-System](https://github.com/physicsilu/Supermarket-Management-System)
- **Original Author**: [physicsilu](https://github.com/physicsilu)
- **Contributors**: physicsilu (Project Lead), Yaswanthbits (Group Member)
- **Features**:
  - Comprehensive database management
  - User interface with JavaFX and JavaSwing
  - SQL-based data operations
  - Project documentation (DBS_Report)
